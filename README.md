# 🏦 Banking Management System (Python + MySQL)

A **Command Line Banking Management System** built using **Python**, **MySQL**, and **Email OTP Verification**.  
This project allows users to create bank accounts, deposit/withdraw money, generate PINs, view account details, and send greeting emails.

---

## 📌 Features

- ✅ Account Creation with Email OTP Verification
- 💰 Deposit Money
- 💸 Withdraw Money (with PIN verification)
- 📄 Mini Statement (View Account Details)
- 🔐 PIN Generation
- 🔄 Change PIN
- 📂 Export All Account Details (Admin Access)
- 📧 Send Greeting Emails to All Users

---

## 🛠 Technologies Used

- **Python**
- **MySQL**
- **PyMySQL**
- **SMTP (Email Sending)**
- **Regex Validation**
- **HTML Email Templates**
